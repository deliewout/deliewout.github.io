Fire Escape Dispatch

Fire Escape Dispatch is a game I developed as part of a group project for Fluvia, the firehouse of Kortrijk.

Project Details

This project involved teamwork in a group setting and simulating a real work week, where we had to work from 8.30 - 17.00 and apply feedback from our client and teachers. We used Unreal C++ for the core game development, HacknPlan for organizing tasks and tracking progress, and Perforce for version control to manage our codebase effectively.

What I Learned

from this project i learned how it is to work in a small work environment. It has also helped to deepen my understanding in Unreal C++.
